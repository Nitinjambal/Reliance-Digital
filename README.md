# Reliance-Digital
# Note:- Run Json-Server before start website   ex- npm run server (in terminal)
## Reliance Digital: It is an online electronic shopping store in India from where you can buy products in a reasonable price. 


# Features
* Navbar
* Products Sections
* Footer
* Cart Page
* Login/Signup


# Tech Stack
* React
* Router-dom
* Chakra-Ui
* NPM 
* Json-Server
* Chakra Componants
